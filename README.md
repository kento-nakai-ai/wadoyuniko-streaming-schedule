# バイブコーディング（Vibe Coding）ローンチプロジェクト 🚀

**「アイデアを、最速でカタチに。」**

## 📌 プロジェクト概要

**バイブコーディング**は、プログラミング経験の少ない方や過去に挫折した方を対象とした、AIを活用した革新的な開発教育プログラムです。`Cursor`や`Claude Code`などの最新AI開発ツールを使用し、自然言語での対話を通じて直感的にアプリケーション開発を学べるスクールです。

### 🎯 プロジェクトの目標
- **売上目標**: 3,000万円
- **受講者目標**: 100名
- **価格設定**: 
  - バイブコーディングスクール: 30万円
  - セット版（プロンプトエンジニアリング込み）: 50万円
- **ローンチ期間**: 2025年8月28日〜30日

### 🎪 主催・協力
- **主催**: wadoyuniko（和道ユニコ）
- **コンテンツ提供**: ユニコーン社（既存80名の受講生実績あり）
- **形態**: 共同ローンチプロジェクト

## ✨ 特徴

- **実績あるコンテンツ**: 既に80名の受講生が成果を上げている実証済みプログラム
- **AI開発特化**: 複雑なコード記述ではなく、AIとの対話でアプリ開発
- **実践重視**: 実際のツール開発を通じた ハンズオン学習
- **コミュニティ**: Discordでの継続的なサポートとメンバー交流
- **収益化支援**: 学んだスキルでクライアントワークや副業に活用

## 🛠️ 技術スタック

### フロントエンド
- **Next.js 15** (App Router) - Reactベースのフルスタックフレームワーク
- **TypeScript** (strict mode) - 型安全性を保証
- **Tailwind CSS + shadcn/ui** - ユーティリティファーストCSS + Radix UI
- **React 19** - UI構築ライブラリ
- **Zustand** - 軽量状態管理
- **next-themes** - ダーク/ライトテーマ切り替え

### バックエンド・データベース
- **Supabase** - PostgreSQLデータベース + 認証システム
- **Next.js API Routes** - サーバーサイド処理
- **Server Actions** - フォーム処理とサーバー操作

### UI/UX
- **Radix UI** - アクセシブルなプリミティブコンポーネント
- **Lucide React** - アイコンライブラリ
- **React Hook Form + Zod** - フォーム管理とバリデーション
- **Recharts** - グラフとチャート
- **Sonner** - トースト通知

### 開発・品質管理
- **Jest + Testing Library** - テストフレームワーク
- **ESLint + Prettier** - コード品質とフォーマット
- **TypeScript** - 静的型チェック

### AI統合
- **Claude API** (Sonnet 4, Opus 4.1) - AI開発支援
- **OpenAI API** (o3) - MCP経由での高度な推論

## 🔧 環境変数 / Environment Variables

開発に必要な環境変数を `.env.local` に設定:

```bash
# Supabase (必須)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url  
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# AI API Keys (サーバーサイドのみ)
CLAUDE_API_KEY=your_claude_api_key
OPENAI_API_KEY=your_openai_api_key

# アプリケーション
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
