# インフォグラフィック設計書

## 基本仕様
- **縦型**: 1080x1920px (9:16) - Instagram/TikTok用
- **横型**: 1920x1080px (16:9) - YouTube/Twitter用
- **正方形**: 1080x1080px (1:1) - Instagram投稿用
- **解像度**: 300dpi
- **フォーマット**: PNG/SVG

## コンテンツ別インフォグラフィック

### 1. バイブコーディング学習フロー
**目的**: 受講から実践までの流れを視覚化
**構成要素**:
- セミナー参加 → 環境構築 → 実践プロジェクト → 応用展開
- 所要時間: 3日間 → 1週間 → 1ヶ月 → 継続
- 成果物: 知識 → 環境 → 作品 → スキル

### 2. AI開発ツール比較チャート
**目的**: Claude Code vs 他ツールの優位性
**比較項目**:
- 学習コスト
- 開発速度
- カスタマイズ性
- コミュニティサポート
- 料金体系

### 3. 30万円vs50万円コース比較
**目的**: プラン選択の判断材料
**比較要素**:
- 基本内容（共通）
- プロンプトエンジニアリング（50万円のみ）
- サポート期間
- 特典数
- ROI計算例

### 4. 受講生成功事例フロー
**目的**: 具体的な成功パターンの可視化
**事例パターン**:
- フリーランス独立
- 社内DX推進
- スタートアップ創業
- 副業収入獲得

### 5. AI開発スキルマップ
**目的**: 習得スキルの体系化
**スキル分類**:
- 基礎技術（プログラミング、Git等）
- AI特化技術（プロンプト、エージェント設計等）
- ビジネススキル（企画、営業等）
- 応用技術（自動化、統合等）

### 6. 投資回収シミュレーション
**目的**: 経済的価値の数値化
**計算要素**:
- 受講料: 30万円 or 50万円
- 想定収入増: 月額10-50万円
- 回収期間: 1-5ヶ月
- 年間ROI: 200-1000%

### 7. 学習時間配分ガイド
**目的**: 効率的な時間割の提示
**時間配分**:
- セミナー参加: 3日間 (24時間)
- 復習・実践: 週10時間 × 4週
- 応用開発: 週5時間 × 継続
- コミュニティ活動: 週2時間

### 8. テクノロジーロードマップ
**目的**: 技術トレンドと学習順序
**技術要素**:
- 2025年: Claude Code, MCP
- 2026年: o3モデル, エージェント統合
- 2027年: 自律型AI開発
- 2028年以降: AGI時代対応

## デザインガイドライン

### ビジュアル要素
- **アイコン**: Lucide React準拠
- **イラスト**: フラットデザイン、親しみやすい
- **チャート**: 見やすい配色、明確な数値
- **フロー**: 矢印、ステップ番号で導線明確化

### カラーシステム
- **プライマリ**: #3B82F6 (情報・説明)
- **セカンダリ**: #10B981 (成功・ポジティブ)
- **アクセント**: #F59E0B (注意・重要)
- **ニュートラル**: #6B7280 (補助情報)

### タイポグラフィ
- **見出し**: Noto Sans JP Bold, 32-48px
- **本文**: Noto Sans JP Regular, 16-24px
- **数値**: Inter Bold, 強調表示
- **キャプション**: Noto Sans JP Light, 12-14px
