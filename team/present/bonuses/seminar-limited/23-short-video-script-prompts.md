### プロンプト01: プラットフォーム最適化アナライザー

```
以下の動画企画を{プラットフォーム}に最適化してください。

【基本情報】
- 動画テーマ: {テーマ}
- ターゲット: {ターゲット層}
- 目的: {認知拡大/フォロワー獲得/売上向上}
- 予算: {制作予算}

【プラットフォーム別最適化】

TikTok最適化:
- 推奨動画長: 15-60秒
- アルゴリズム対策: 完視聴率重視、初速重要
- 人気要素: 音楽トレンド、チャレンジ、エフェクト
- ハッシュタグ戦略: トレンド+ニッチの組み合わせ
- 投稿時間: 19-21時がピーク

YouTube Shorts最適化:
- 推奨動画長: 15-60秒
- アルゴリズム対策: クリック率とリテンション重視
- 人気要素: 教育系、ハウツー、ストーリー
- タイトル戦略: 検索を意識したキーワード
- サムネイル: 縦型でインパクト重視

Instagram Reels最適化:
- 推奨動画長: 15-30秒
- アルゴリズム対策: シェア率とコメント重視
- 人気要素: 美的センス、ライフスタイル、インスピレーション
- ハッシュタグ戦略: 関連性の高いタグ10-15個
- ストーリーズ連携: Reelsの予告を投稿

【最適化提案】
1. コンテンツ構成の調整
2. 投稿タイミングの提案
3. ハッシュタグ戦略
4. エンゲージメント施策
5. 継続投稿計画

各プラットフォームで最大の効果を得る戦略を提案してください。
```

## 📈 エンゲージメント向上プロンプト {#エンゲージメント}

### プロンプト11: コメント誘発装置

```
{動画内容}でコメントが殺到する仕組みを設計してください。

【コメント誘発の種類】

## 1. 質問型誘発
動画内容に関連した質問を投げかける

例:
- 「あなたはどっち派？コメントで教えて！」
- 「同じ経験した人いる？」
- 「他に良い方法知ってる人いませんか？」

設計:
- 質問のタイミング: {動画の何秒目}
- 質問の内容: {具体的な質問文}
- 画面表示: {テキストの見せ方}

## 2. 議論誘発型
賛否が分かれる内容を提示

例:
- 「これについてどう思いますか？」
- 「賛成の人は👍、反対の人は👎」
- 「異論がある人はコメントください」

設計:
- 論点の提示: {何について議論させるか}
- 立場の明確化: {自分のスタンス}
- 反対意見への対応: {炎上回避策}

## 3. 共感誘発型
多くの人が経験していることを題材に

例:
- 「これ分かる人RT」
- 「同じ人〜🙋‍♀️」
- 「あるある度100%」

設計:
- 共感ポイント: {何に共感させるか}
- 体験談の提示: {具体的エピソード}
- 参加の促し: {コメント参加の方法}

## 4. 教え合い誘発型
知識や経験の共有を促す

例:
- 「他の方法知ってる人教えて」
- 「みんなのおすすめを聞かせて」
- 「経験者の話を聞きたい」

設計:
- 知識の種類: {何について教え合うか}
- 投稿者の姿勢: {学ぶ態度を示す}
- 感謝の表現: {コメントへの返信方針}

【コメント促進戦略】
動画内での仕込み:
- {秒数}秒目: 「コメントで教えて」
- {秒数}秒目: 選択肢の提示
- {秒数}秒目: 質問の投げかけ

投稿後の対応:
- 最初の1時間でのアクティブな返信
```
