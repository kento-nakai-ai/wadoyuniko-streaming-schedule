# Live（配信）チーム

## チーム概要

わどユニコプロジェクトの配信・ライブストリーミングを担当するチームです。YouTube配信、セミナーのライブ配信、配信環境の構築・運用、配信コンテンツの企画・制作を行います。

## 主な責任範囲

- YouTube ライブ配信の企画・実行
- セミナーライブ配信の技術サポート
- 配信環境・機材の構築・運用
- 配信スケジュールの管理
- 配信コンテンツの企画
- OBS・配信ツールの設定・運用
- 配信アーカイブの管理

## 配信環境・機材

### 配信ソフトウェア
- **メイン**: OBS Studio
- **サブ**: StreamLabs OBS
- **配信プラットフォーム**: YouTube Live

### 必要機材
- **カメラ**: 4K Webカメラ または DSLR
- **マイク**: コンデンサーマイク
- **照明**: LED パネル
- **配信PC**: 高性能CPU・GPU必須

### 配信設定
- **解像度**: 1920x1080 (Full HD)
- **フレームレート**: 30fps
- **ビットレート**: 6000 kbps
- **音声**: 44.1kHz, 128kbps

## 配信フロー

### 配信前準備（60分前）
1. ✅ 機材・環境確認
2. ✅ OBS シーン設定
3. ✅ 音声・映像テスト
4. ✅ 配信タイトル・説明設定
5. ✅ SNS 告知投稿

### 配信開始（30分前）
1. ✅ YouTube Live 開始
2. ✅ 待機画面表示
3. ✅ BGM 再生開始
4. ✅ チャット監視開始

### 配信中
1. ✅ 進行・司会
2. ✅ チャット対応
3. ✅ 画面共有・デモ
4. ✅ 時間管理

### 配信後
1. ✅ アーカイブ確認
2. ✅ サムネイル設定
3. ✅ 概要欄更新
4. ✅ SNS フォローアップ

## KPI・目標

### 視聴数目標
- **ライブ視聴者**: 平均100名以上
- **アーカイブ再生**: 1動画1000回以上
- **チャンネル登録者**: 月500人増加

### エンゲージメント
- **チャット参加率**: 30%以上
- **いいね率**: 10%以上
- **コメント率**: 5%以上

### セミナー配信特別目標
- **同時視聴者**: 300名以上
- **参加維持率**: 80%以上（最後まで視聴）
- **個別相談申込**: 配信経由50件以上

## トラブルシューティング

### よくある問題
1. **音声トラブル**: マイク設定・音量調整
2. **映像トラブル**: カメラ接続・解像度設定
3. **配信遅延**: ネットワーク・エンコード設定
4. **OBS クラッシュ**: 設定リセット・再起動

### 緊急時対応
- **配信トラブル**: バックアップ環境への切替
- **機材故障**: 予備機材の準備
- **ネットワーク障害**: モバイル回線での配信

## 連絡先

- **報告先**: PMチーム（中井健登）
- **連携チーム**: マーケティング、Design、Content
- **緊急時**: 配信チャンネル・直接連絡
