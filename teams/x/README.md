## バズ投稿戦略

### チーズバーガー戦略
- **概念**: 魅力的な外観（タイトル）+ 満足感（中身）
- **実装**: キャッチーなフック + 価値ある情報
- **例**: 「AIで月100万稼ぐ方法」→ 具体的な手法解説

### サトリ記法
- **概念**: 読者の気づき・共感を誘発する表現
- **手法**: 体験談、失敗談、意外な発見の共有
- **効果**: 高いエンゲージメント、RTの促進

### AIスレッド投稿
- **フォーマット**: 連続ツイートでの情報提供
- **内容**: AI開発の手順、ツール比較、成功事例
- **頻度**: 週2-3回の投稿
