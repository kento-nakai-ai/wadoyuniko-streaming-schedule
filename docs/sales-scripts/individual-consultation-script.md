### 9.1 基本姿勢
- **顧客の成功を第一に考える**
- **価格ではなく価値で判断してもらう**
- **急かすのではなく、納得してもらう**
- **断られても相手を責めない**

4. **希少性の演出**: 限定性を適切に伝える

### 9.3 NGワード・行動
