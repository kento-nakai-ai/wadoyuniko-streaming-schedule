# 進捗ダッシュボード

## 実行内容
1. /docs/tasks/ 内の全チケットをスキャン
2. Todo項目の完了率を計算
3. ガントチャート風の表示
4. ブロッカーの可視化
5. KPIメトリクスの算出

## 出力形式
```

================================

タスク別状況:

002: ████████░░░░  66% 🔄
003: ████░░░░░░░░  33% 🔄
004: ░░░░░░░░░░░░   0% ⏳

- 002_データベース設計

```

## KPI算出
- ベロシティ: 完了タスク数/期間
- バーンダウン: 残タスク推移
- リードタイム: 着手→完了時間
- ブロッカー率: ブロック中/全タスク
